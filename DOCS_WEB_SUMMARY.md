macroplannerAI — Generador automático de menús nutricionales

macroplannerAI genera menús saludables y personalizados mediante IA para aplicaciones web y móviles. El sistema combina una interfaz React (Vite) con un servicio backend en Cloudflare Workers que construye menús en JSON a partir de listas de alimentos predeterminadas y un modelo de lenguaje. Está diseñado para integrarse fácilmente con frontends y para desplegarse con `wrangler`.

Beneficios clave:

- Menús rápidos y consistentes basados en objetivos nutricionales (pérdida de grasa, ganancia de masa, mantenimiento).
- Control total sobre los alimentos permitidos mediante archivos JSON locales.
- Respuesta en formato JSON lista para consumir por la UI o APIs.
